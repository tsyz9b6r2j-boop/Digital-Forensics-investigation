# Network Security Lab

## Overview
This project demonstrates basic network security assessment using VirtualBox, Kali Linux, and network scanning tools.

## Objectives
- Identify active devices in a network
- Perform port scanning and service detection
- Analyze network traffic
- Identify potential security risks

## Tools Used
- Kali Linux
- Nmap
- Wireshark
- VirtualBox

## Methodology

### 1. Network Discovery
Performed subnet scanning to identify active hosts.

### 2. Port Scanning
Used Nmap to identify open ports and running services.

### 3. Traffic Analysis
Captured and analyzed packets using Wireshark.

## Key Findings
- Open ports identified on target machine
- SSH and HTTP services detected
- Network traffic successfully captured and analyzed

## Skills Demonstrated
- Network scanning
- Traffic analysis
- Cyber security fundamentals
- Linux command line
